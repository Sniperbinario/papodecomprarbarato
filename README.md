# Papo de Comprar Barato — Landing Page

Conteúdo gerado automaticamente por ChatGPT a pedido do usuário.

## Estrutura do projeto

- `index.html` — página principal (pronta para deploy)
- `css/styles.css` — estilos organizados e responsivos
- `produtos.json` — arquivo com as ofertas (edite esse arquivo para atualizar as ofertas)
- `produtos.js` — script que carrega `produtos.json` e monta os cards
- `favicon.ico` — ícone (placeholder)

## Como subir no Vercel (recomendado)
1. Crie conta em https://vercel.com
2. Faça `New Project` → `Import` → `Upload` → selecione os arquivos extraídos do ZIP
3. Deploy automático — pronto!

## Observações
- Substitua os links `placeholder-affiliate` pelos seus links de afiliado reais antes de publicar.
- As imagens são hospedadas no Unsplash; se preferir, troque para imagens da loja ou CDN própria.
- Para atualizar ofertas, edite apenas o arquivo `produtos.json`.
